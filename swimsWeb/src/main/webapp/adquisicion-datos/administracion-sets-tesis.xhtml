Diríjase al sítio http://repositorio.utn.edu.ec/oai/request?verb=ListSets para conocer los identificadores y nombres de los sets puestos a disposición por la Universidad Técnica del Norte.

Considere que:

  • Un Set tiene un formato similar a: "col_123456789_40" y se encuentra entre corchetes a la derecha del título o nombre del Set.
  • Usted puede elegir el nombre a tomar por el Set registrado, este nombre puede diferir del título anotado en la Web del Repositorio de la Universidad Técnica del Norte.
  • El identificador debe ser único, es decir un set puede ser registrado una sola vez.
